# 迁移到 Scoop：Windows 软件管理的革命性升级

如果你还在用浏览器手动下载 exe / msi 安装包，还在为软件更新东奔西跑，那么 `Scoop` 将彻底改变你的 Windows 使用体验。Scoop 是 Windows 上的命令行包管理器，让安装、更新、卸载软件像 Linux 的 apt 或 macOS 的 Homebrew 一样简单。

## 1. 为什么选择 Scoop？

### 🚀 极简安装体验

- **一行命令安装**：无需打开浏览器、找官网、点下一步
- **无需管理员权限**：默认安装到用户目录，干净无污染
- **无弹窗、无捆绑**：纯净安装，零广告零骚扰

### 🎯 一体化管理

- **统一更新**：`scoop update *` 一键升级所有软件
- **版本管理**：可锁定版本、切换版本
- **依赖自动处理**：安装时自动解决运行时依赖

### 🔒 可重复部署

- **新机一键还原**：导出已安装列表，新机器一键重建环境
- **脚本友好**：完美融入自动化工作流

## 2. 安装 Scoop

### 一键安装（推荐）

打开 PowerShell，执行：

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex
```

### 添加常用 Bucket

```powershell
scoop bucket add extras
scoop bucket add versions
```

### 验证安装

```powershell
scoop --version
```

## 3. 用 Scoop 安装现代播放器

告别 Microsoft Store 的龟速下载和各家官网散装安装包，所有播放器一行命令搞定。

### 安装 Screenbox（Windows 首推）

```powershell
scoop install screenbox
```

Fluent Design + Mica 材质，Windows 11 完美融入，LibVLC 内核保证格式兼容性。

### 安装 MPV（极简硬核首选）

```powershell
scoop install mpv
```

顶级画质渲染，支持 HDR、插帧、AV1 硬件加速，脚本扩展无上限。

### 安装 SMPlayer（自定义爱好者）

```powershell
scoop install smplayer
```

主题丰富，Material Dark 皮肤 + MPV 后端，高度可定制。

### 保留 VLC 作兜底（可选）

```powershell
scoop install vlc
```

网络流媒体（RTSP/HLS）场景下的可靠后备。

## 4. 播放器调优：装完即用，调优更佳

### Screenbox – Fluent 极简优化

```powershell
scoop install screenbox
```

安装后推荐设置：

- Settings → Appearance → Dark mode（或 System）
- Enable gesture controls（触控板/鼠标手势调音量/进度）
- Picture-in-Picture + Mini mode（右键 → Mini player）

效果：像 Windows 11 原生 Photos，但支持所有格式。

### MPV – 硬核画质调优

```powershell
scoop install mpv
```

在 `%APPDATA%\mpv\mpv.conf` 中添加：

```
vo=gpu-next
hwdec=auto
profile=gpu-hq
interpolation=yes
```

效果：顶级 HDR 渲染 + 极低资源占用。

### SMPlayer – 自定义到极致

```powershell
scoop install smplayer
```

安装后：Options → Preferences → Interface：

- Icon set：Papirus / Breeze Dark
- Style：Modern / Fusion Dark
- Multimedia Engine → mpv（切换为 MPV 后端）

## 5. 统一更新：一行命令保持最新

```powershell
# 更新 Scoop 自身
scoop update

# 更新所有已安装软件
scoop update *

# 只更新指定软件
scoop update screenbox
```

## 6. 从手动安装迁移

### 卸载旧版软件

先卸载系统中手动安装的旧版播放器，再用 Scoop 重新安装，避免冲突：

```powershell
# 安装后 Scoop 统一接管版本管理
scoop install screenbox
scoop install mpv
scoop install smplayer
```

### 导出已安装列表

```powershell
# 导出列表，方便新机还原
scoop export > scoop-packages.json

# 新机还原
scoop import scoop-packages.json
```

## 7. 性能对比

| 操作场景       | 手动安装方式                  | Scoop 方式               | 体验提升   |
| -------------- | ----------------------------- | ------------------------ | ---------- |
| **安装新软件** | 找官网 → 下载 → 点安装 → 重启 | `scoop install xxx`      | 效率飞跃   |
| **更新软件**   | 逐个打开检查更新              | `scoop update *`         | 一键完成   |
| **切换版本**   | 手动卸载再装旧版              | `scoop reset xxx@版本号` | 秒级切换   |
| **新机还原**   | 逐个下载重装                  | `scoop import`           | 分钟级还原 |
| **卸载干净**   | 残留注册表/文件               | `scoop uninstall xxx`    | 零残留     |

## 8. 常见问题解决

### Q: 找不到想要的软件？

```powershell
# 搜索软件
scoop search screenbox

# 添加更多 bucket
scoop bucket add nirsoft
scoop bucket add nonportable
```

### Q: 安装失败或网络问题？

```powershell
# 使用代理
scoop config proxy 127.0.0.1:7890

# 清除代理
scoop config rm proxy
```

### Q: 如何锁定版本不自动更新？

```powershell
scoop hold screenbox
scoop unhold screenbox  # 解除锁定
```
